Hexo介绍

Hexo是一个快速、简洁且高效的博客框架

官网：Hexo官方网站链接

特点：

  1. 超快速度：Node.js 所带来的超快生成速度，让上百个页面在几秒内瞬间完成渲染。
  2. 支持Markdown：Hexo 支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。
  3. 一键部署：只需一条指令即可部署到 GitHub Pages, Heroku 或其他平台。
  4. 插件和可扩展性：强大的 API 带来无限的可能，与数种模板引擎（EJS，Pug，Nunjucks）和工具（Babel，PostCSS，Less/Sass）轻易集成。

SSH Key配置

在完成了上述操作后，其实就可以通过hexo部署命令将博客部署到Github了，但是每次推送发布都需要输入账号密码，比较麻烦，所以我们需要配置一下SSH Key（觉得还是每次输入账号密码比较好的可以忽略这一步）

SSH密钥生成之后在Mac中默认在个人的.ssh里面（注意：.ssh是隐藏文件，若你未开启查看隐藏文件，则需要按Command + Shift +句点（CMD + Shift +。）来打开查看隐藏文件，当不需要再查看隐藏文件时，重复这个操作即可）

打开id_rsa.pub文件，直接复制其内容，或使用终端运行 eval "$(ssh-agent -s)"获取其内容然后复制下来

登陆Github，点击头像，选择Settings → SSH and GPG keys → New SSH Key

将复制的密钥放入Key中即可，Title可随意起
